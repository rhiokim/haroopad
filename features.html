Haroopad

is a markdown enabled document processor for creating web-friendly documents.

You can author various formats of documents such as blog article, slide, presentation, report, and e-mail as if experts did.

Haroopad gives you same experiences in editing regardless of the platform you are working on. It runs on all three major operating systems—Windows, Mac OS X, and Linux.

Main Features

Supported Cross Platform

Haroopad support (Ubuntu, 32/64 bit) Windows 7/8, Mac OS X, of Linux.

Github Flavored Markdown + Extensions

Not only Haroopad supports Markdown(GFM), but also has ability to build extensions for making your own features.

Exporting to various formats(Coming soon)

You can export your documents for Wordpress, Tumblr, Mediaiki, EPub, ReStructured Text, RTF, and so on. (At the moment, it only supports HTML and PDF)

Beautiful Themes, Skins and UI Components

Haroopad supports modern themes in editing area, skins for transformed documents, UI components for interactive experiences and syntax hightlighting for various programming languages. With these features, you can generate more legible documents.

Easy Imports Contents (Coming soon)

More and more, we tend to save our documents on YouTube, Blogs, Slideshare, Flickr, and so on. Now, you can easily embed those contents into your documents.

Blogging and Mailing New!

Your documents could be transformed into Blog system, Wordpress, Evernote and Tumblr, without any effort. Furthermore, you can send your documents as a email.

Sending document as email with excellent readability

It would be great if recipients could see transfered web document, written in Markdown, as you have seen.

Everything of headers, images, tables, citations and even syntax highlighted codes in your documents could be delivered to your recipients as it has seen to you.

You will experiece that Haroopad is more powerful than usual WYSWYG editors of mail clients. This feature help us to share common ideas and to reduce misunderstandings from email communications.

Posting document to your favorite blog service

This is derived feature from mailing.

As you already know, Markdown is so versatile for documenting and has been widely spread to many different environments.

Althought it lets you compose emails, blog articles, manuals, memos, and even diary, it would be more powerful when it is converted into HTML format than itself.

With this converting feature, Haroopad provides posting documents into Tumblr.

We are going to support more blog systems in the future.
