Haroopad

is a markdown enabled document processor for creating web-friendly documents.

You can author various formats of documents such as blog article, slide, presentation, report, and e-mail as if experts did.

Haroopad gives you same experiences in editing regardless of the platform you are working on. It runs on all three major operating systems—Windows, Mac OS X, and Linux.


Markdown? Learn Haroopad

Main Features

Supports Cross Platforms

Haroopad runs on Linux, Windows7/8, Mac OS X.

Multimarkdown + Extensions

Not only Haroopad supports Multimarkdown, but also has ability to build extensions for making your own features.

Exporting to various formats(Coming soon)

You can export your documents for Wordpress, Tumblr, Mediaiki, EPub, ReStructured Text, RTF, and so on. (At the moment, it only supports HTML and PDF)

Beautiful Themes, Skins and UI Components

Haroopad supports modern themes in editing area, skins for transformed documents, UI components for interactive experiences and syntax hightlighting for various programming languages. With these features, you can generate more legible documents.

Easy Imports Contents (Coming soon)

More and more, we tend to save our documents on YouTube, Blogs, Slideshare, Flickr, and so on. Now, you can easily embed those contents into your documents.

Blogging and Mailing(Coming soon)

Your documents could be transformed into Blog system, Wordpress and Tumblr, without any effort. Furthermore, you can send your documents as a email.
